# Search

MkDocs ships with a built-in `search` plugin, enabled by default in most configurations (including this book).
It indexes page content at build time and provides client-side search in the rendered site.

## Tips

- Keep headings descriptive: search results tend to surface section titles.
- Prefer canonical terms in headings and first paragraphs to improve relevance.
- If you rename files or headings, rebuild locally to confirm search results still point to the right pages.
