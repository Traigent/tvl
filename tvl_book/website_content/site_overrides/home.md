# TVL Intelligent Textbook

![Traigent Robot Trainer](./img/robot-trainer.svg){ width="720" }

*Transform AI agents from experimental to production-ready with governed configuration.*

Welcome to the Traigent Labs companion for the Tuned Variable Language (TVL). This site packages the book,
hands-on labs, walkthroughs, and playground resources that teach engineers how to:

- Capture configuration as **TVL specs** instead of brittle YAML scattered across services.
- Use **TVO** (Traigent Variable Optimizer) to search safely under multi-objective budgets.
- Validate datasets with **DVL** before promoting any configuration.
- Keep promotion trails auditable with deterministic exports and CI gates.

The material blends narrative chapters, runnable examples from the `Traigent` repository, and an interactive
Streamlit playground. Start with the [TVL Intelligent Textbook](./tvl/index.md) chapter map or jump directly
to a [hands-on lab](./tvl/resources.md) to run an optimization.
